Convergence curves of normalized squared error versus iterations for the PGRR algorithm and its momentum-accelerated variants under different momentum parameters (μ). The baseline (μ=0.0) corresponds to the standard PGRR method without momentum. Momentum values μ = 0.5, 0.7, 0.9, and 0.99 correspond to the proposed method with increasing momentum, as indicated in the legend. All methods were evaluated on a sparse signal recovery task (MNIST image reconstruction from random linear measurements) with identical initialization and measurement noise. Objective error values were recorded every 3000 iterations. The y-axis is shown in logarithmic scale to highlight convergence behavior. Higher momentum values lead to faster and more stable convergence, with μ=0.99 demonstrating the lowest final error.
