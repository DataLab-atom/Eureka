Convergence error comparison curves of the PGRR algorithm and the proposed algorithm. The horizontal axis represents the number of iterations, and the vertical axis represents the error (logarithmic scale). The purple solid line represents the error change of the PGRR algorithm, and the blue dashed line represents the error change of the proposed algorithm. The dashed curve on the right (yellow) represents the error difference between the two algorithms (PGRR error minus the error of the proposed algorithm). The shaded area indicates the improvement range where the PGRR error exceeds the error of the proposed algorithm. The green dashed line marks the iteration number (5,000) with the largest error difference. The figure shows that the proposed method outperforms the PGRR algorithm at all iterations, with the largest improvement highlighted. This indicates that the introduction of the momentum term effectively improves convergence performance and error reduction speed.
